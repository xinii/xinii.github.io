为 Fish 和 Emacs 添加 apt 仓库

想安装 Emacs26 或 Fish 3.0 却发现 apt 提示已经安装最新版本？来试试本文的方法吧！

June 3, 2020 Xin Li

将要使用的命令 

  add-apt-repository

没这个命令？先试试这个 

  apt install software-properties-common

安装过程

注册仓库 

  add-apt-repository ppa:kelleyk/emacs
  apt-add-repository ppa:fish-shell/release-3

刷新仓库 

  apt update

安装 

  apt install emacs26
  apt install fish
